# M4 Core Bot for Immie Server
Welcome to the repository that hosts the **guts and flesh** of the M4 Core Bot in the Immie Discord server!

## Technology
We use **Python and Cogs** to run our bot in a Virtual Environment. To keep development fluid, it supports:
* **Hot-reloading:** Swap out code on the fly.
* **Self-pulling:** Automatically update from the repository to any branch.

## Branches
We maintain two primary branches to keep things stable:
* **Main:** The stable, production-ready code.
* **Canary:** Where the magic (and the bugs) happen. All changes are tested and audited here before being merged to Main.

> **Note:** You may see additional branches created for Pull Requests. These are temporary and will be deleted once they are merged and closed.
